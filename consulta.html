Gostaria de pedir um minuto da atenção de vocês para falar sobre o sistema :)

O Sistema de cadastro de alunos, utiliza um recurso de Web Storage, ou seja, é utilizado para cadastrar os dados, um recurso local do próprio navegador, o Local Storage.

Sendo assim, é recomendado que você limpe o seu cache de navegação para manter este local limpo e tirar o máximo de proveito da aplicação, evitando possíveis bugs durante o uso.

Existem inúmeras ferramentas para limpeza do cache, uma que utilizo e recomendo bastante é o CCleaner, da Piriform, que é gratuito e super simples de se utilizar, pode confiar ;D

Deixarei o link de download logo abaixo e o link para um tutorial rápido de uso da ferramenta, são só 2 minutinhos...
